RX5808-Div Firmware Configurator

Firmware management for RX5808 Diversity FPV drone video receivers

System Requirements

  • Chrome, Edge, or Opera browser with Web Serial API support
  • USB connection to RX5808-Div device
  • CH340/CP2102 USB drivers (Windows)

Key Features

Wireless Control

ExpressLRS integration via ESP-NOW protocol for remote channel selection

VTX Band Swap

Configurable R/L band remapping for compatibility with various VTX firmware

Multi-Band Support

Complete coverage of all 6 bands (A, B, E, F, R, L) with 8 channels each
